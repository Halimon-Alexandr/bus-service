Документація до нашого API

Адреса API сервера: http://mog-pod.pp.ua

Отримання інформації про автобуси за назвою зупинки

Запит

GET /bus/info/

Параметри запиту:

  • bus_stop_name (опціонально): Назва зупинки автобуса (назву писати можна не повністю). За замовчуванням пустий рядок.
  • offset (опціонально): Зміщення по маршрутам. Попередній, наступний (за замовчуванням 0).

Відповідь

Якщо успішно:

  • next_bus_info: Об'єкт, що містить інформацію про наступний автобус.
    • bus_stop: Назва зупинки автобуса.
    • direction: Напрямок руху автобуса.
    • bus_number: Номер автобуса.
    • arrival_time: Час прибуття автобуса у форматі "години:хвилини".
    • time_left: Час до прибуття автобуса у форматі "години хвилини" або "хвилини", якщо години відсутні.
    • full_schedule: Повний графік руху автобусів по знайденій зупинці.
  • stops: Список зупинок, якщо знайдено більше однієї зупинки за схожою назвою.

Якщо зупинку не знайдено, повертається відповідь зі статусом 404 та повідомленням "Зупинку не знайдено".

Якщо сталася помилка на сервері, повертається відповідь зі статусом 500 та повідомленням про помилку.
